# 存储管理

告别磁盘耗尽。在容器持续运行的同时扩容仓库，将已删除的数据块归还给存储池，然后让机器自动完成这两项工作。

## 观看教程

![Tutorial: Storage management](/assets/tutorials/tutorial-storage-management.cast)

## 一个共享池，仓库按需增长

![一个共享池，仓库按需增长](/img/tutorials/tutorial-storage-management/slide-1.svg)

每个仓库都是共享数据存储池中的一个文件。仓库可以在线扩容；在仓库内部删除数据本身并不会释放空间，**trim** 才是将数据块归还给存储池的操作。
