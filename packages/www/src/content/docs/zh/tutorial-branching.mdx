# 分支：提交与回滚

像管理代码一样管理仓库版本。**commit** 将 fork 冻结为不可变镜像；**branch** 为其命名；**checkout** 将任意提交还原为全新可写 fork。本教程中，我们提交一个已知良好的状态，然后误删一张真实的 PostgreSQL 表，再用秒级回滚找回每一行数据。

## 模型说明

![提交、分支、检出](/img/tutorials/tutorial-branching/slide-1.svg)

提交永不改变：它们甚至拒绝挂载。Fork 从不等待：checkout 是近乎即时的 reflink 克隆。完整的心智模型详见[分支指南](/zh/docs/repo-branching)。
