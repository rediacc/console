# Быстрый старт

Добро пожаловать в Rediacc. Этот раздел проведет вас через всё необходимое для развертывания зашифрованной изолированной инфраструктуры на ваших собственных серверах.

## Начало работы

Следуйте этим страницам по порядку:

1. **[Требования](/ru/docs/requirements)** — Поддерживаемые операционные системы и предварительные требования для вашей рабочей станции и удаленных серверов.

2. **[Установка](/ru/docs/installation)** — Установка CLI `rdc` на Linux, macOS или Windows (WSL2).

3. **[Пошаговое руководство](/ru/docs/guide)** — Создание контекста, добавление машин, создание зашифрованных репозиториев и развертывание сервисов.

4. **[После установки](/ru/docs/post-installation)** — Настройка автозапуска, описание конфигурации контекста и устранение распространенных неполадок.

5. **[Инструменты](/ru/docs/tools)** — Синхронизация файлов, SSH-терминал, интеграция с VS Code, обновления CLI и диагностика.

## Что такое Rediacc?

Rediacc развертывает контейнеризированные сервисы на удаленных серверах, которые вы контролируете. Всё шифруется в состоянии покоя с помощью LUKS, каждый репозиторий получает собственный изолированный Docker-демон, а вся оркестрация происходит по SSH с вашей рабочей станции.

Никаких облачных аккаунтов. Никаких SaaS-зависимостей. Ваши данные остаются на ваших серверах.
