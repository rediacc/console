# Esimese repositooriumi loomine

Rediacc repositoorium on üks krüpteeritud fail sinu serveris. Ühenda see ja saad kausta, millel on oma Dockeri deemon ja oma rakenduse andmed, eraldatuna kõigest muust serveris.

Mõtle sellele kui tootmiskeskkonna USB-mälupulgale: puhkeolekus on see fail, käivitamisel server.

## Vaata juhendvideot

![Tutorial: Creating your first repository](/assets/tutorials/tutorial-create-repo.cast)

## Kettale kirjutatud fail, ühendamisel keskkond

![Encrypted file mounts as an isolated folder](/img/tutorials/tutorial-create-repo/slide-1.svg)

Kettal on see üksik krüpteeritud kujutis. Ühendamisel saad:

- Pühendatud Dockeri deemoni (eraldi hosti omast)
- Rakenduse andmed krüpteeritud mahus
- Loopback IP-d, mis ei põrku serveri millegi muuga

Repositooriumid liiguvad. Kopeeri üks masinate vahel, varunda see või tee sellest fork (forkid on peaaegu hetkelised ja konstantse ajaga, 100 GB repositoorium forkeritakse sama kiiresti kui 1 GB repositoorium). Iga repositoorium on eraldatud kõigist teistest repositooriumitest samal serveril.
