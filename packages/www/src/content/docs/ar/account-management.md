### المؤسسات

عند التسجيل، يقوم Rediacc تلقائيا بانشاء مؤسسة لك. مؤسستك هي الحاوية الرئيسية لجميع الموارد -- الاجهزة والمستودعات والاشتراكات واعضاء الفريق.

![Registration Flow](/img/account-registration-flow.svg)

تحتوي كل مؤسسة على:
- اسم فريد (يكون البريد الالكتروني بشكل افتراضي)
- خطة اشتراك (تبدا بـ COMMUNITY)
- فريق افتراضي (ينضم اليه جميع الاعضاء تلقائيا)

### الاعضاء والادوار

تدعم المؤسسات ثلاثة ادوار:

![Role Hierarchy](/img/account-role-hierarchy.svg)

| الدور | القدرات |
|-------|---------|
| **Owner** | تحكم كامل: الفوترة، نقل الملكية، ادارة جميع الاعضاء والفرق |
| **Admin** | دعوة وازالة الاعضاء، انشاء وادارة الفرق، الغاء رموز API |
| **Member** | عرض بيانات المؤسسة، انشاء رموز API، الوصول الى الفرق المعينة |

دعوة الاعضاء:
```bash
# From the portal: Organization > Members > Invite
# Or via API
```

عند ازالة عضو، يتم الغاء رموز API ورموز config storage الخاصة به تلقائيا.

### الفرق

تتيح لك الفرق تحديد نطاق الموارد داخل المؤسسة. تبدا كل مؤسسة بفريق افتراضي.

![Team Structure](/img/account-team-structure.svg)

ادوار الفريق:
- **Team Admin**: يمكنه اضافة/ازالة اعضاء الفريق داخل الفريق
- **Member**: يمكنه الوصول الى الموارد المحددة بنطاق الفريق

يحصل مالكو ومديرو المؤسسة تلقائيا على حق الوصول الى جميع الفرق دون الحاجة الى عضوية صريحة.

### الاشتراكات والخطط

يقدم Rediacc اربع خطط:

| الخطة | الاجهزة | تراخيص المستودعات/شهر | الميزات |
|-------|---------|------------------------|---------|
| COMMUNITY | 2 | 500 | اساسي |
| PROFESSIONAL | 10 | 2,000 | مجموعات الصلاحيات، اولوية قائمة الانتظار |
| BUSINESS | 25 | 5,000 | Ceph، تحليلات متقدمة، سجل التدقيق |
| ENTERPRISE | غير محدود | غير محدود | علامة تجارية مخصصة، حساب مخصص |

![Subscription Flow](/img/account-subscription-flow.svg)

تبدا جميع الخطط بفترة سماح مدتها 3 ايام. يتم تتبع تنشيطات الاجهزة لكل فريق ويتم تحريرها تلقائيا بعد عدم النشاط.

### الفوترة

يمكن لـ **مالك** المؤسسة فقط ادارة الفوترة:
- انشاء جلسة دفع Stripe لترقيات الخطة
- الوصول الى بوابة فوترة Stripe لتغيير طريقة الدفع
- طلب استرداد الاموال ذاتيا (خلال 14 يوما، مع فترة انتظار 30 يوما)
