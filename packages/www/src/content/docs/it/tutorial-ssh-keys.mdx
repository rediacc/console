# Configurazione della Chiave SSH

Le chiavi SSH ti permettono di accedere ai tuoi server senza password. Si tratta di SSH standard, indipendente da `rdc`: generi una chiave sul tuo laptop, poi la autorizzi su ogni server.

## Guarda il tutorial

![Tutorial: Configurazione della chiave SSH](/assets/tutorials/tutorial-ssh-keys.cast)

## Come funziona

![Genera una chiave, poi autorizzala su ogni server](/img/tutorials/tutorial-ssh-keys/slide-1.svg)

1. **Genera** una chiave SSH sul tuo laptop. Una volta, per sempre.
2. **Autorizzala** su ogni server con `ssh-copy-id`. Ripeti per ogni nuovo server.

## Passo 1: Genera una chiave

Se hai già una chiave che vuoi usare, passa al passo successivo. Altrimenti:

<TutorialStep number={1} />

`ed25519` è il default moderno: piccolo, veloce e ben supportato.

## Passo 2: Autorizza il tuo primo server

<TutorialStep number={2} />

Sostituisci `<user>` e `<server-ip>` con l'utente SSH e l'IP del tuo server. Ti verrà chiesta la password del server un'ultima volta. Dopo questo, l'autenticazione con password non sarà più necessaria.

## Passo 3: Autorizza gli altri server

<TutorialStep number={3} />

Ripeti `ssh-copy-id` per ogni macchina che gestisci. È tutto ciò di cui SSH ha bisogno. Nel prossimo tutorial, `rdc` usa questa chiave automaticamente.
