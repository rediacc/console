# 설치

`rdc` 설치는 세 단계입니다: 설치 페이지를 열고, 운영 체제를 선택하고, 명령을 터미널에 붙여넣습니다. 보통 1~2분이면 완료됩니다.

## 튜토리얼 보기

![Tutorial: Installation](/assets/tutorials/tutorial-installation.cast)

## 세 단계

![Three steps overview](/img/tutorials/tutorial-installation/slide-1.svg)

1. [설치 페이지](/ko/install)를 엽니다.
2. 운영 체제를 선택합니다.
3. 설치 명령을 복사하여 터미널에 붙여넣습니다.

## 1단계: CLI 설치

<TutorialStep number={1} />

설치 페이지에서 올바른 명령을 생성합니다. Windows(PowerShell)의 경우 동등한 한 줄 명령은 다음과 같습니다:

```powershell
iwr -useb https://www.rediacc.com/install.ps1 | iex
```

> `time` 접두사는 명령이 얼마나 걸렸는지 출력하는 셸 트릭입니다. 이 시리즈 전체에서 각 단계의 실제 속도를 확인할 수 있도록 사용합니다. 선택 사항입니다. 원하지 않으면 생략해도 됩니다.

## 2단계: 설치 확인

<TutorialStep number={2} />

`rdc doctor`를 실행하여 Node, SSH, 그 외 `rdc`의 의존성을 점검하고 부족한 부분을 보고합니다.

## `rdc`가 노트북에 있는 이유

![rdc on your laptop, renet on the server](/img/tutorials/tutorial-installation/slide-2.svg)

`rdc`는 노트북의 CLI입니다. 서버는 `renet`이라는 별도의 구성 요소를 실행하며, `rdc`가 SSH를 통해 프로비저닝하고 구동합니다. 서버에 직접 SSH로 접속할 필요가 없습니다. `rdc`가 대신 처리합니다.

다음 두 튜토리얼에서 이것을 제대로 설정합니다.
