# 모니터링

앱이 배포되고, 실행 중이고, 백업되었습니다. 이제 모든 것이 건강하게 유지되도록 합니다. `rdc`는 노트북에서 상태, 컨테이너, 저장소 등 모든 서버의 전체 모습을 제공합니다.

## 상태 확인: 시스템 정보

<TutorialStep number={1} />

시스템 가동 시간, 디스크 사용량, 스토리지 상태를 표시합니다. 문제가 있으면 알려줍니다.

## 컨테이너

<TutorialStep number={2} />

각 컨테이너의 이름, 상태, 헬스, CPU, 메모리와 어떤 저장소가 소유하는지 확인할 수 있습니다.

## 저장소

<TutorialStep number={3} />

크기, 마운트 상태, Docker 상태, 디스크 사용량과 함께 모든 저장소를 표시합니다.

## 한 번에 모든 것 확인

<TutorialStep number={4} />

시스템 정보, 저장소, 컨테이너 모두 하나의 명령으로. 필터 없이 사용하는 `query` 명령은 전체 모습을 반환하고, `--system`, `--containers`, `--repositories`, `--services`, `--network`, `--block-devices`로 해당 섹션만 좁힐 수 있습니다.
